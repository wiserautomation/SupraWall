- **Description:** Deterministic security layer for Vercel AI SDK applications. Intercept tool calls, enforce budget caps, and ensure EU AI Act compliance at the edge.

## Key Features
1. **Edge Runtime Interception**: Sub-1ms policy evaluation integrated directly into Next.js.
2. **Tool Governance**: Deterministic "Allow/Deny/Request Approval" logic for LLM-driven tools.
3. **Audit Trails**: Detailed logs of every agent action for security auditing.
4. **Budget Caps**: Prevent runaway costs from autonomous loops.
