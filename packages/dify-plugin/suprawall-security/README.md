# SupraWall Security for Dify

SupraWall is the multi-security layer for AI agents. This plugin allows your Dify agents to intercept tool calls, enforce custom security policies, and generate EU AI Act compliant audit trails with zero latency.

## Features
- **Prompt Injection Defense**: Detect and block malicious intent in agent inputs.
- **Tool Integrity**: Ensure tool calls match authorized schemas and behavioral policies.
- **PII Redaction**: Automatically identify and mask sensitive user data.
- **Audit Logging**: Comprehensive forensic trails of every agent decision.

## Installation
1. Obtain an API Key from the [SupraWall Dashboard](https://supra-wall.com).
2. Configure your `SUPRAWALL_API_KEY` in the Dify plugin settings.
3. Wrap your security-critical tools with the SupraWall provider.
